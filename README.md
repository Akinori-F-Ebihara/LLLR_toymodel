# LLLR: Toy-model
 A simple 3-layer fully connected network performing the density ratio estimation using the loss for log-likelihood ratio estimation (LLLR).

## Tested Environment
- tensorflow 2.0.0
- CUDA 10.0
- cuDNN 7.6.4.38
